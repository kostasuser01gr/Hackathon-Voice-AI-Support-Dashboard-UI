# Screenshot Checklist

Store screenshots under `docs/` using these filenames:

1. `docs/screenshot-dashboard.png`
2. `docs/screenshot-history.png`
3. `docs/screenshot-session-detail.png`
4. `docs/screenshot-settings.png`
5. `docs/screenshot-integrations.png`
6. `docs/screenshot-actions.png`
7. `docs/screenshot-status.png`

Required content:
- Dashboard with processed output cards visible.
- History list with at least one session.
- Session detail opened from `/history/[id]`.
- Settings diagnostics panel showing env + observability.
- Integrations job status after one run.
- Actions board showing approval state filter.
- Status page with health links.
